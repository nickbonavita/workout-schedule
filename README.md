# Workout Schedule Website

A beautiful, interactive workout schedule website built with HTML, CSS, and JavaScript. Features five workout categories with unique pastel color schemes.

## Features

- **5 Workout Sections**: Push, Pull, Legs, Cardio, and Core
- **Interactive Tabs**: Smooth switching between workout categories
- **Pastel Color Schemes**: Each section has its own unique, beautiful color theme
- **Workout Details**: Each exercise includes targeted muscle groups
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## GitHub Pages Setup

1. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select `main` branch and `/root` folder

2. **Access Your Site**:
   - Your site will be available at: `https://nickbonavita.github.io/workout-schedule/`
   - GitHub Pages automatically serves `index.html` as the homepage

## Files

- `index.html` - Main HTML structure with all workout content
- `styles.css` - Styling with pastel color schemes and responsive design
- `script.js` - JavaScript for interactive tab functionality

## Customization

Feel free to:
- Add or remove exercises
- Modify the pastel colors in `styles.css`
- Adjust the layout or styling
- Add more sections or details to exercises

## Local Testing

Simply open `index.html` in your web browser to view the site locally before deploying.
